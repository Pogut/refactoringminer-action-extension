Choose when RefactoringMiner analyses supported GitHub PR and commit diffs.

Highlight colours

The colour each selected refactoring blinks in, used in every mode, on both PR and commit pages. The left (old) and right (new) sides of the diff blink in different colours so their counterparts are easy to pair up. The outline and off-screen line markers use a slightly darker shade, derived automatically.

How fast a selected refactoring pulses once it has been clicked. At the lowest setting the highlight stays lit constantly and never blinks; each step up shortens the pulse. The preview above follows this setting.

Standalone service

Used only in standalone mode, on commit pages, or repos that don't run the RefactoringMiner GitHub Action, where the extension asks a hosted RefactoringMiner service to analyse the commit. Pages that already have the action's published feed ignore these.

Must send Access-Control-Allow-Origin so the browser can read it.
